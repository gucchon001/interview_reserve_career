When all the results for a request can't be returned, a
nextPageToken
token parameter will be returned in the response. This can be used to paginate the results.
To fetch results for the following page, add
nextPageToken
as a query parameter.
For example:
GET https://timerex.net/api/beta/users/me/teams?nextPageToken=<token>
