APIキーによる認証
TimeRex 日程調整APIを利用する際の認証方法は「OAuth認証」の他に、APIキーによる認証方法があります。
APIキーの作成・確認
TimeRexにログインします。
ダッシュボード > チーム設定 > デベロッパーツール > TimeRex 日程調整API からAPIキーを作成します。
※ APIキー仕様: 64桁の半角英数字
作成されたAPIキーをコピーして使用してください
APIキーの使い方
TimeRex 日程調整APIを利用する際に、リクエストヘッダーに「x-api-key」を設定し送信します。
リクエストヘッダ名
設定値
x-api-key
作成されたAPIキー
使用例:
curl -i -X GET \
-H "x-api-key:2Hy3qF4HFmYxqPlkkGfILhQ2Ps3mcQqvHnwxxxxxxxxxxxxxxxxxxxxxx" \
'https://timerex.net/api/beta/user/me/teams'
