日程調整カレンダーウィジェット（サイトへの埋め込み機能）

日程調整カレンダーウィジェットを利用すると、WebサイトにTimeRexの日程調整カレンダーを埋め込むことができ、Webサイト上で日程調整ができるようになります。

利用条件

全てのプランで利用可能です。

基本的な利用方法

日程調整カレンダーの設定確認ページの「日程調整カレンダーの埋め込み」に表示されるコードをWebサイトのBodyタグ内に追加すると、Webサイトに日程調整カレンダーを埋め込むことができます。

利用例

実際にWEBページに埋め込んでいただいた際のサンプルページは以下をご覧ください。

https://timerex.net/blog/update/201109-embed

TimeRexウィジェットの色変更例は以下をご覧ください。

https://timerex.net/blog/update/210218-color

日程調整カレンダーウィジェットへのデータ連携

Webサイトで管理している会員情報等を日程調整カレンダーウィジェットに連携することができます。データ連携方法は「設問項目の自動入力」と「パラメータ連携」の2つがあり、それぞれ、埋め込みコードの一部を変更することで簡単に実現可能です。

設問項目の自動入力（データ連携1）

日程調整時の設問項目に顧客の名前やメールアドレス等を自動入力しておくことができます。

※存在しないパラメータが渡された場合、最大25個のurl_paramsの上限にカウントされます。

サンプルコード

<!-- Begin TimeRex Widget --> 
<div id="timerex_calendar" data-url="https://timerex.net/s/xxxxxxxxxx/xxxxxxxx"></div>
<script id="timerex_embed" src="https://asset.timerex.net/js/embed.js"></script>
<script type="text/javascript">
    TimerexCalendar({
        'guest_company' : 'Mixtend Inc.',
        'guest_name' : 'Tomohiro Kitano',
        'guest_email' : 'xxxxxxxx@test.com',
        'guest_comment' : 'Hello',
        'locale': 'en',
        '1234567890abc_1': '01-01-1990' // Date of birth
    });
</script>
<!-- End TimeRex Widget -->

パラメータ

名前データ型必須備考
guest_companystringfalse標準設問項目の「会社名」に入る値
guest_namestringfalse標準設問項目の「名前」に入る値
guest_emailstringfalse標準設問項目の「メールアドレス」に入る値
guest_commentstringfalse標準設問項目の「コメント」に入る値
localestringfalse表示する言語を指定できます。
  • ja: 日本語
  • en: 英語
<custom_form_field_id>stringfalse任意の設問項目に入る値を指定できます。（プレミアムプランのみで利用できます）

パラメータ名となる<custom_form_field_id>は、設問項目の編集画面から確認することができます。詳しくはこちら。
primary_colorstringfalseウィジェットのキーカラーをカラーコードで指定できます。（プレミアムプランのみで利用できます）

カラーコードで指定例
  • #00000000（8桁)
  • #000000（6桁）
  • #000（3桁）
  • 00000000（#なし）
  • 000000（#なし）
  • 000（#なし)
disable_logostringfalseウィジェットのTimeRexロゴを非表示にできます（プレミアムプランのみで利用できます）
  • true: ロゴを非表示
  • false: ロゴを表示
disable_title_hyperlinkstringfalseウィジェットの日程調整カレンダー名のハイパーリンクを無効にできます（プレミアムプランのみで利用できます）
  • true: ハイパーリンクを無効
  • false: ハイパーリンクを有効

パラメータ連携（データ連携2）

ページ上には表示しない値を日程調整カレンダーウィジェットに連携することができます。

会員を識別するIDや流入元等の把握に利用できます。

サンプルコード

<!-- Begin TimeRex Widget --> 
<div id="timerex_calendar" data-url="https://timerex.net/s/xxxxxxxxxx/xxxxxxxx"></div>
<script id="timerex_embed" src="https://asset.timerex.net/js/embed.js"></script>
<script type="text/javascript">
    TimerexCalendar({
        "url_params": {
            "user_hash": "abcdefg1234567",
            "campaign_id": "Osaka_SMB"
        }
    });
</script>
<!-- End TimeRex Widget -->

パラメータ

名前データ型必須備考
url_paramsArrayfalse最大25個のパラメータを渡すことができます（25個を超えるとエラーが表示されます）。
パラメータはURLエンコードする必要があります。
パラメータにネストしたオブジェクト/配列を設定することはできません。

※Webhookを利用する場合
受け取るリクエストは、パラメータ単位でオブジェクトの配列に変換されます。

日程調整カレンダーウィジェットの読み込みタイミングの制御

埋め込む先のWebページ等の問題によりウィジェットの読み込みタイミングを制御したい場合、以下のようにコードを変更し、TimerexCalendar() 関数を呼び出すことで制御できます。

手順1：ウィジェットコードをBody内に追加

Body内のウィジェットを表示したい部分に以下のコードを追加します。

<!-- Begin TimeRex Widget -->
<div id="timerex_calendar" data-url="https://timerex.net/s/demo.team/xxxxxxxx"></div>
<script id="timerex_embed" src="https://asset.timerex.net/js/embed.js"></script>
<!-- End TimeRex Widget -->

手順2：関数を呼び出すコードを追加

ウィジェットを読み込みたいタイミングでTimerexCalendar() 関数を呼び出します。

<button onclick="TimerexCalendar();">Load</button>

Callback機能

ウィジェットのライフサイクルで特定のイベントが発生すると、ホストページはメッセージを受信することが出来ます。

Callback types

名前説明
onLoad()ウィジェットの読み込みが完了したことを受信します。
onFormOpen()予約フォームが表示されたことを受信します。
onBookingComplete()予約が完了したことを受信します。

サンプルコード

<!-- Begin TimeRex Widget --> 
<div id="timerex_calendar" data-url="https://timerex.net/s/xxxxxxxxxx/xxxxxxxx"></div>
<script id="timerex_embed" src="https://asset.timerex.net/js/embed.js"></script>
<script type="text/javascript">
    TimerexCalendar({
      'onLoad': function () {
        console.log('Widget Loaded');
      },
      'onFormOpen': function () {
        console.log('Booking form opened');
      },  
      'onBookingComplete': function () {
        console.log('Booking completed');
      }, 
    });
</script>
<!-- End TimeRex Widget -->

Callbackを利用してGoogle広告でコンバージョン計測する場合のサンプルコード

<!-- Begin TimeRex Widget --> 
<div id="timerex_calendar" data-url="https://timerex.net/s/xxxxxxxxxx/xxxxxxxx"></div>
<script async src="https://www.googletagmanager.com/gtag/js?id=xxxxxxxx"></script>
<script>
    window.dataLayer = window.dataLayer || [];
    function gtag(){dataLayer.push(arguments);}
    gtag('js', new Date());
</script>
<script id="timerex_embed" src="https://asset.timerex.net/js/embed.js"></script>
<script type="text/javascript">
    TimerexCalendar({
        'onLoad': function () {
            console.log('Widget Loaded');
        },
        'onFormOpen': function () {
            console.log('Booking form opened');
        },  
        'onBookingComplete': function () {
            console.log('Booking completed');
            gtag('event', 'conversion', {
                'send_to': 'xxxxxxxx',
                'value': 1.0,
                'currency': 'USD'
            });
        }, 
    });
</script>
<!-- End TimeRex Widget -->

※Google広告でコンバージョン計測する場合は、gtag()内のsend_toパラメータは'AW-CONVERSION_ID/CONVERSION_LABEL'が入ります。

※Google Analyticsでコンバージョン計測する場合は、gtag('js', new Date());の1行下にgtag('config', 'トラッキングID');が必要です。また、gtag()内のsend_toパラメータは任意です（入力する場合、'トラッキングID'が入ります）。

詳しくはこちらをご参照ください。

予約完了後にUTMパラメータを送信するサンプルコード

<!-- Begin TimeRex Widget --> 
<div id="timerex_calendar" data-url="https://timerex.net/s/xxxxxxxxxx/xxxxxxxx"></div>
<script id="timerex_embed" src="https://asset.timerex.net/js/embed.js"></script>
<script async src="https://www.googletagmanager.com/gtag/js?id=xxxxxxxx"></script>
<script>
    window.dataLayer = window.dataLayer || [];
    function gtag(){dataLayer.push(arguments);}
    gtag('js', new Date());


    // お客様のWebページURLからutm_parametersを取得する例
    const queryParams = new URLSearchParams(window.location.search);
    const utm_source = queryParams.get('utm_source');
    const utm_medium = queryParams.get('utm_medium');
    const utm_campaign = queryParams.get('utm_campaign');
</script>
<script type="text/javascript">
    TimerexCalendar({
      'onBookingComplete': function () {
        console.log('Booking completed');
        gtag('event', 'utm_paramter', {
          'send_to': 'xxxxxxxxxxxx',
          "utm_medium": utm_medium,
          "utm_source": utm_source,
          "utm_campaign": utm_campaign
        });
      }, 
    });
</script>
<!-- End TimeRex Widget -->
