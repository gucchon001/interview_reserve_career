Push-Notifications

この記事は日本語でもご覧いただけます。

You can subscribe to push notifications using /watch endpoints for supported resource types.

Handling push notification

On receiving a push notification you must return a 20x response as soon as possible. If a 20x response is not received the push notification will be retried two more times.

Troubleshooting

If you are unable to receive push notifications, ensure the following

  • The url is accessible from the internet
  • You have a valid SSL certificate
  • localhost and ngrok url's are not supported
